## 🔍 Cómo Buscar Documentos

### Por Tipo de Problema:

- **Error 403/Autenticación** → Busca "FIX-403-" o "SOLUCION-FINAL-AUTH"
- **Problemas con API** → Busca "FIX-API" o "SOLUCION-ERROR"
- **Problemas de datos** → Busca "FIX-MAPEO", "FIX-NOMBRE", "FIX-DATOS"
- **Problemas visuales** → Busca "FIX-JUGADORES", "FIX-MENSAJES", "FIX-ESCUDOS"

### Por Funcionalidad:

- **Alineaciones** → "CREAR-ALINEACION", "MIS-ALINEACIONES", "VER-ALINEACIONES"
- **Partidos** → "CREAR-PARTIDOS", "SELECCION-LIGA"
- **Comentarios** → "SISTEMA-COMENTARIOS", "INSTRUCCIONES-COMENTARIOS"
- **Login/Registro** → "PAGINA-LOGIN", "PAGINA-REGISTRO", "COMO-FUNCIONA-LOGIN"
- **Votos** → "VOTOS-", "MIGRACION-VOTOS"

### Por Prefijo:

- **FIX-** → Solución a un problema específico (20 docs)
- **SOLUCION-** → Solución detallada (11 docs)
- **IMPLEMENTACION-** → Implementación de feature (3 docs)
- **RESUMEN-** → Resumen de sesión/trabajo (8 docs)
- **GUIA-** → Guía de uso (2 docs)
- **COMO-** → Explicación de funcionamiento (3 docs)
- **DIAGNOSTICO-** → Análisis de problema (2 docs)
- **MEJORA-** / **MEJORAS-** → Optimizaciones (5 docs)
